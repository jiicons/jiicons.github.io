Installation

Install from CDN

There is jsDelivr CDN available for quickly integrated with your web page.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/jiicons@0.2.3/jiicons.css">

After setup in your HTML, you can start to use the icons by following our examples.
